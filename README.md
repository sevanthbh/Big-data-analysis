# Big-data-analysis Big Data Analysis Dashboard is a Streamlit-powered web application designed for handling and visualizing large datasets efficiently. With a sleek dark mode UI, the dashboard allows users to generate, process, and analyze big data in real time. It leverages Dask for scalable data processing and provides interactive visualizations using Pandas and Matplotlib. Users can adjust dataset sizes, explore data distributions, and gain insights instantly.
